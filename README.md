## Módulo HTML&CSS 
### Aula01
#### Tópicos de Aula
- Introdução ao HTML:
- O que é HTML e como ele funciona.
- A história e evolução do HTML.
- O HTML 5
- A estrutura básica de um documento HTML (tags, elementos, atributos).
---
#### Preparando o ambiente:
- Editores de código (VS Code, Sublime Text, etc.).
- Configuração do ambiente de desenvolvimento.
- Criação do primeiro arquivo HTML.
- Visualização de páginas HTML em navegadores.
- Usando o Live Server.
---
#### Estrutura da página:
- Tags básicas: `<head>, <body>, <html>, <!DOCTYPE>`;
- Títulos (`<h1> a <h6>`);
- Parágrafos (<p>);
- Div e Span
- Quebras de linha (<br/>);
- Linhas horizontais (<hr/>);
- Listas ordenadas (<ol>);
- Listas não ordenadas (<ul>);
- Listas de definição (<dl>);
- Links e atributos (<a>, href, target);
- Imagem (<img>);
- Atributos de imagem (src, alt, width, height);
- Tabelas (<table><tr><td><th>);
- Atributos de tabelas (colspan, rowspan);
